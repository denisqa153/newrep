Инструкция по работе с гитхабом
1. Установить гит на МАКОС
2. После установки нужно указать username и email, сделать это нужно через команды:
*git config --global user.name '4len4lenov'
*git config --global email.'zzz@mail.rap'
4. $ git config --list - чтобы вывести содержимое файла с ником и почтой для проверки
5. Сделать папку гит-репозиторием:
*перейти в папку и ввести= git init(при репозы должны быть отдельными друг от друга)
-прим: для удаления-
$ cd <папка с репозиторием> # перешли в папку
$ rm -rf .git # удалили подпапку .git
(ключ -r (от англ. recursive — «рекурсивно») позволяет удалять папки вместе с их содержимым;
ключ -f (от англ. force — «заставить») избавит вас от вопросов вроде «Вы точно хотите удалить этот файл? А этот? И этот тоже?».)
6. Для проверки состояния репоза git status
7. Можно добавлять файлы 
8. После их добавления их нужно "взять в кадр" - подготовить к сохранению через = git add 'название файла' 
= git add --all - все файлы
= git add . - всю папку
9. "Делаем снимок" - сохраняем =git commit -m 'текст что изменено'
10. История комитов git log

Регистрация на гитхабе
1.Создать новый репоз
2.Связать ssh ключи
3. Из домашней директории cd ~ проверить наличие ssh директории ls -la .ssh/
4. Для генерации ssh ключа использовать:
$ ssh-keygen -t rsa -b 4096 -C "электронная почта, к которой привязан ваш аккаунт на GitHub"
5. Указать место хранения ключей, например домашнюю директорию:
> Enter a file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter] 
6. Затем указать кодовую фразу или оставить поле пустым
7. Проверить создался ли ключ: ls -a ~/.ssh
8. Скопировать rsa.pub и приложить в гитхаб в настройках в shh ключ
$ pbcopy < ~/.ssh/id_rsa.pub
9. Сравнить правильность ключа с $ ssh -T git@github.com 
10. Связываем удаленный и лоакльный репоз, создаем репоз в гитхабе переходим в удаленный репоз копируем shh в нужной гит папке выполняем:
$ git remote add origin git@github.com:%ИМЯ_АККАУНТА%/first-project.git 
11. Убедиться что репозы связаны git remote -v
12. Запушить в репоз git push
